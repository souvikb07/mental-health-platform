# User Interview Script

## Intro

Thanks for testing this. This is an early hackathon prototype, not therapy or diagnosis. Please do not share anything you are not comfortable sharing.

## Before using

1. What would normally make you look for mental health support?
2. What makes it hard to know where to start?
3. Have you ever felt like something was wrong but you could not name it?

## During use

Ask the user to think aloud while using:

- landing page
- onboarding
- chat
- clarity map
- resources

## After use

1. Did this help you feel clearer?
2. Did any part feel unsafe, too clinical, or too vague?
3. Were the next steps realistic?
4. Would you use this before talking to a therapist/counsellor?
5. What would make you trust it more?

## Ratings

```txt
Clarity: 1-5
Helpfulness: 1-5
Felt safe: yes/no
Would recommend: yes/no
```

## Quote permission

Can we use an anonymized quote from your feedback in the hackathon demo?
