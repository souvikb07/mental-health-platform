# Validation Plan

## Phase 1 validation question

Does a short AI-guided clarity session help users feel clearer about what they are experiencing and what to do next?

## Validation methods

1. Ask 5-10 target users to run through the MVP.
2. Collect clarity and helpfulness ratings.
3. Ask for one sentence: "What did this help you understand?"
4. Ask what felt unsafe, generic, or wrong.
5. Capture anonymized feedback quotes for demo.

## Core metrics

- completion rate
- clarity rating
- helpfulness rating
- felt-safe response
- feedback quote quality

## Qualitative questions

- What were you feeling before using this?
- Did the Clarity Map describe your situation fairly?
- Did the next steps feel realistic?
- Would you share this with a friend?
- What would make this trustworthy?

## Red flags

- Users think it diagnosed them.
- Users think it replaces therapy.
- Users receive vague advice.
- High-risk users are not routed safely.
- Users find resource suggestions fake or unhelpful.
